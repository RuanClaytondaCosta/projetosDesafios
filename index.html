falando sobre a tag p

(Voltar)

A tag p basicamente fala sobre a criação de parágrafos e assim escrever quantas palavras e textos que você quiser,
essa tag é utilízada para estruturar e formatar o seu texto, essa tag delimita onde vai ser o inicio e o fim de um bloco de texto.

falando sobre a tag strong

(Voltar)

A tag strong tem como o objetivo de dar enfase nas palavras que estão dentro da tag as deixando em negríto

falando sobre a tag id

(Voltar)

a tag Id se trata de identificar um elemento unico e depois o manípular a sua vontade, a maneira de identíficar o elemento é utilizando por meio de palavras que você escrever, e por isso você utilizando
essa mesma palavra no identificador você o utiliza para manipula-lo da forma que quiser

falando sobre a tag small

(Voltar)

a tag small tem como objetivo
deixar tudo que estiver dentro da tag pequeno exemplo small

falando sobre a tag blockquote

(Voltar)

a tag blockquote é utilizada para mostrar que a frase do texto que você colocou dentro da tag é uma citação.

falando sobre a tag abbr

(Voltar)

a tag abbr tem como objetivo falar sobre que esta na abreviação como DIO que significa Digital Inovation One

falando sobre a tag font

(Voltar)

a tag font tem como objetivo alterar o estilo do texto conforme quiser como a cor verde que acabei de colocar

falando sobre a tag sup

(Voltar)

a tag sup tem como objetivo de exibir um conteudo considerado como subscrito mas que geralmente usado para simbolizar um caractere ou numeros pequenos acima da linha de texto como por exemplo a potenciação de 2 2

falando sobre a tag sub

a tag sub é similar ao sup mas ao contrario da linha acima agora é a linha abaixo como por exemplo numeros atomicos da tabela periódica como H 2o

falando sobre html

(Voltar)

o html é uma linguagem de de marcação utilizada para criar e projetar paginas web, você cria-las e manipuladas do jeito que você quiser a partir de tags.
